## Run Locally

Clone the project

```bash
  git clone https://github.com/jaysontolentino/react-auth-api.git
```

Go to the project directory

```bash
  cd react-auth-api
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn dev
```

## Tech Stack

**Server:** Node, Express, Prisma
